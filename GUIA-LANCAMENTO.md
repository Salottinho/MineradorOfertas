## 5.3 Metricas para acompanhar

### O que olhar e onde

No Ads Manager, personalize as colunas clicando em **Colunas** > **Personalizar colunas** e adicione:

- CTR (taxa de cliques no link)
- CPC (custo por clique no link)
- CPR (custo por resultado)
- Impressoes
- Alcance
- Frequencia

### Metas para cada metrica

| Metrica | Meta | O que significa |
|---------|------|----------------|
| **CTR (link)** | > 1.5% | Seu criativo esta chamando atencao |
| **CPC (link)** | < R$1,50 | Cada clique custa pouco |
| **CPA (venda)** | < R$25,00 | Custo por cliente. Produto custa R$37, entao lucro = R$12+ |
| **ROAS** | > 1.5x | Para cada R$1 investido, volta R$1,50+ |
| **Frequencia** | < 3.0 | Cada pessoa viu o anuncio menos de 3 vezes (nao saturou) |
| **Taxa quiz > venda** | > 8% | De cada 100 pessoas que fazem o quiz, 8+ compram |

### Quando MATAR um criativo (pausar)

- Gastou **R$50+ sem nenhuma venda** naquele conjunto → Pausar imediatamente
- CTR abaixo de **0.8% apos 1.000 impressoes** → Hook fraco, pausar
- CPC acima de **R$3,00 consistentemente** → Pausar
- Frequencia acima de **3.5** → Publico saturado, pausar nesse conjunto
- 3 dias sem melhora na tendencia → Pausar

### Quando NAO mexer

- Antes de 1.000 impressoes (dados insuficientes)
- Antes de 48 horas (fase de aprendizado do algoritmo)
- Se o CPA esta alto mas o CTR esta bom (o problema pode ser no quiz ou checkout, nao no anuncio)

### Quando ESCALAR (aumentar investimento)

- CPA consistente abaixo de **R$20 por 3+ dias** → Aumentar orcamento 20-30%
- CTR acima de **2% com CPC abaixo de R$1** → Duplicar conjunto com orcamento maior
- ROAS acima de **2x estavel por 3 dias** → Hora de escalar agressivamente

### Como escalar

- **Escala vertical:** Aumentar orcamento do conjunto vencedor em 20-30% a cada 2 dias (NUNCA mais de 30% de uma vez — reseta o aprendizado)
- **Escala horizontal:** Duplicar conjunto vencedor com novos interesses ou publicos semelhantes (lookalike)
- **Novos criativos:** Criar variacoes do criativo vencedor (mesmo hook, visual diferente) toda semana

Quando a GoDaddy liberar o dominio rendasilenciosa.com.br, execute a Fase 1.2 (DNS) e depois atualize TODOS os links:

1. **Instagram bio:** trocar link para `https://rendasilenciosa.com.br`
2. **Anuncios (se ja estiverem rodando):** editar URL de destino para `https://rendasilenciosa.com.br`
3. **Kiwify - pagina de obrigado:** trocar para `https://rendasilenciosa.com.br/obrigado.html`
4. O arquivo `docs/CNAME` ja contem `rendasilenciosa.com.br`, entao o GitHub Pages vai responder automaticamente
