CHECKLIST 30 DIAS DO AFILIADO

Metodo Renda Silenciosa

COMO USAR ESTE CHECKLIST

Este checklist foi feito para ser seu guia pratico durante os primeiros 30 dias como afiliado de produtos fisicos. Cada dia tem tarefas especificas com tempo estimado e investimento necessario.

Como aproveitar ao maximo:

  1. Imprima este checklist ou salve no celular para acesso rapido
  2. Marque cada tarefa conforme for concluindo — o progresso visual motiva
  3. Siga a ordem proposta — cada semana constroi sobre a anterior
  4. Nao pule etapas, mesmo que parecam simples
  5. Ao final dos 30 dias, voce tera sua primeira estrutura de vendas funcionando

O objetivo e claro: ao final de 4 semanas, voce tera uma maquina de vendas basica rodando com trafego pago, presenca no Pinterest, e participacao em comunidades — tudo sem aparecer.

Vamos comecar.

SEMANA 1 — FUNDACAO (Montar toda a estrutura)

DIA 1 — CONTAS E ACESSOS

[ ] Criar conta na Monetizze (monetizze.com.br)
[ ] Criar conta na Hotmart (hotmart.com)
[ ] Criar conta no Facebook Business Manager (business.facebook.com)
[ ] Criar conta Business no Pinterest (pinterest.com/business/create)
[ ] Criar conta no Canva (canva.com)
[ ] Anotar todos os logins em lugar seguro
Tempo estimado: 40 minutos
Investimento: R$0

DIA 2 — ESCOLHER PRODUTO

[ ] Acessar Monetizze > Loja > Produtos Fisicos
[ ] Filtrar por "Maior temperatura"
[ ] Analisar os 5 melhores com o checklist de 8 pontos
[ ] Escolher 1 produto principal
[ ] Solicitar afiliacao
[ ] Copiar e salvar seu link de afiliado
[ ] Estudar a pagina de vendas do produto (ler inteira)
[ ] Anotar: publico-alvo, beneficios principais, objecoes comuns
Tempo estimado: 1 hora
Investimento: R$0

DIA 3 — CRIAR MATERIAIS

[ ] Abrir Canva e buscar templates de "Facebook Ad" e "Pinterest Pin"
[ ] Criar 3 imagens de anuncio para Facebook (1080x1080)
[ ] Escrever 3 textos de anuncio diferentes (gancho + beneficio + CTA)
[ ] Criar 5 pins para Pinterest (1000x1500)
[ ] Salvar tudo numa pasta organizada
Tempo estimado: 2 horas
Investimento: R$0

DIA 4 — SUBIR PRIMEIRA CAMPANHA

[ ] Acessar Gerenciador de Anuncios do Facebook
[ ] Criar campanha: objetivo Trafego
[ ] Configurar publico (idade, genero, interesses)
[ ] Subir primeiro anuncio (imagem + texto + link)
[ ] Definir orcamento: R$20/dia
[ ] Publicar campanha
[ ] NAO MEXER por 48h
Tempo estimado: 1 hora
Investimento: R$20

DIA 5 — PINTEREST

[ ] Criar 5 paineis tematicos no Pinterest
[ ] Publicar os 5 pins criados no Dia 3
[ ] Criar mais 5 pins (total 10 publicados)
[ ] Escrever descricoes com palavras-chave
Tempo estimado: 1 hora
Investimento: R$0

DIA 6 — GRUPOS E COMUNIDADES

[ ] Buscar e entrar em 5 grupos de Facebook do nicho escolhido
[ ] Ler as regras de cada grupo
[ ] Fazer 1 comentario util em cada grupo (sem links)
[ ] Criar um grupo no Telegram: "[Nicho] — Ofertas do Dia"
[ ] Convidar 10-20 contatos para o grupo do Telegram
Tempo estimado: 45 minutos
Investimento: R$0

DIA 7 — PRIMEIRA ANALISE

[ ] Verificar metricas do Facebook Ads (impressoes, cliques, CTR)
[ ] Anotar: CPC (custo por clique), CTR (taxa de clique), alcance
[ ] Verificar se houve vendas na Monetizze
[ ] Criar mais 5 pins e publicar no Pinterest
[ ] Postar primeira oferta no grupo do Telegram
Tempo estimado: 30 minutos
Investimento: R$20 (acumulado na campanha)

RESULTADO ESPERADO SEMANA 1: Estrutura montada, campanha rodando, presenca no Pinterest iniciada.

SEMANA 2 — OTIMIZACAO (Ajustar o que esta funcionando)

DIA 8

[ ] Analisar 4 dias de dados do Facebook Ads
[ ] Se CTR abaixo de 1%: trocar imagem do anuncio
[ ] Se CTR acima de 1% mas sem vendas: verificar se o link esta correto
[ ] Publicar 5 novos pins no Pinterest

DIA 9

[ ] Testar nova variacao de texto no anuncio
[ ] Interagir em 3 grupos do Facebook (comentarios uteis)
[ ] Postar 2 ofertas no grupo do Telegram

DIA 10

[ ] Se o anuncio esta com CPC acima de R$2: pausar e criar novo anuncio
[ ] Se CPC abaixo de R$1: manter e monitorar
[ ] Criar 5 pins novos no Pinterest
[ ] Pesquisar um segundo produto como backup

DIA 11

[ ] Publicar 5 pins no Pinterest
[ ] Postar 2 ofertas no Telegram
[ ] Responder comentarios/mensagens nos grupos

DIA 12

[ ] Revisar orcamento: quanto gastou vs quanto faturou
[ ] Se ROI positivo (faturou mais do que gastou): manter
[ ] Se ROI negativo apos R$60 gastos: mudar criativo ou publico
[ ] Criar 3 novas imagens de anuncio

DIA 13

[ ] Subir nova variacao de anuncio (publico ou criativo diferente)
[ ] Publicar 5 pins no Pinterest
[ ] Postar 2 ofertas no Telegram

DIA 14

[ ] Analise semanal completa:
- Total investido na semana
- Total faturado em comissoes
- Numero de cliques no link
- Numero de vendas
- ROI (faturado / investido)
[ ] Decidir: manter produto atual ou testar outro
[ ] Publicar 5 pins no Pinterest

RESULTADO ESPERADO SEMANA 2: Primeiras vendas (1-5), entendimento de quais criativos funcionam melhor.

SEMANA 3 — ACELERACAO (Aumentar volume)

DIA 15-16

[ ] Aumentar orcamento da campanha vencedora em 25%
[ ] Criar mais 2 variacoes de anuncio
[ ] Publicar 5 pins/dia no Pinterest
[ ] Postar diariamente no Telegram

DIA 17-18

[ ] Duplicar a campanha vencedora com publico ligeiramente diferente
[ ] Entrar em mais 5 grupos de Facebook
[ ] Interagir e recomendar produto quando oportuno
[ ] Continuar ritmo de 5 pins/dia

DIA 19-20

[ ] Analisar qual campanha performa melhor
[ ] Pausar as com ROI negativo, manter as positivas
[ ] Considerar criar um blog simples (WordPress.com gratuito)
[ ] Publicar 1 artigo de review do produto

DIA 21

[ ] Analise semanal completa (mesmo modelo da semana 2)
[ ] Calcular projecao mensal baseada nos dados atuais
[ ] Definir meta para semana 4

RESULTADO ESPERADO SEMANA 3: 5-15 vendas acumuladas, ROI positivo consistente, presenca no Pinterest crescendo.

SEMANA 4 — ESCALA (Rumo a R$3k+/mes)

DIA 22-23

[ ] Se ROI esta positivo: aumentar orcamento mais 25%
[ ] Testar segundo produto com campanha separada (R$15/dia)
[ ] Publicar mais 2 artigos no blog (se criou)
[ ] Manter ritmo de pins e posts no Telegram

DIA 24-25

[ ] Analisar segundo produto: converter ou nao?
[ ] Otimizar anuncios: matar os ruins, escalar os bons
[ ] Convidar mais pessoas para o grupo do Telegram
[ ] Publicar 5 pins/dia

DIA 26-27

[ ] Planejar estrutura para o mes seguinte
[ ] Listar 3 novos produtos para testar
[ ] Automatizar publicacao de pins (usar agendador)
[ ] Criar banco de 10 mensagens de oferta para Telegram

DIA 28-30

[ ] ANALISE FINAL DO MES:
- Total investido no mes
- Total faturado em comissoes
- ROI geral
- Melhores canais (pago vs gratuito)
- Melhor criativo de anuncio
- Projecao para o mes seguinte
[ ] Definir meta de faturamento para o mes 2
[ ] Celebrar suas primeiras vendas como afiliado!

RESULTADO ESPERADO SEMANA 4: R$500-2.000 em comissoes, estrutura funcionando, clareza do que funciona melhor para voce.

METAS PROGRESSIVAS:

  • Mes 1: R$500 - R$2.000 (fase de aprendizado e testes)
  • Mes 2: R$1.500 - R$3.000 (otimizacao e escala)
  • Mes 3: R$3.000 - R$5.000 (multiplos canais rodando)
